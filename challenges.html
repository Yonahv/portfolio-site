10 JavaScript code challenges for beginners

With solutions - A case study

I found these ten challenges on Codecademy. I threw in some html and css to make them functional in the browser.

Below is somewhat of a case study walking through the process of me solving these challenges.

1. Print even numbers from a given array

In this challenge, the requirement was to log all even numbers from 0-10.

To solve it, a simple console.log() would do. (Nice code block font BTW eh?)

Obviously, if I wanted to log even numbers past 10, the most efficient way would be to loop through and only return numbers that were %2==0.

Because I was getting user input, I modified the exercise to accept numbers from the user. Running the function will return only the even numbers. I added an eventListener to allow the user to either press enter or hit the button.

I had some issues with the answer spilling outside of the solution container. I had to use some css word-wrap and overflow properties to fix it.

10. Return the number of vowels in a string

For this, I first declared a variable with an array of all the vowels. I then lopped through the user input and used the .includes() array method to compare the user input to the vowels variable. If a letter in the input was equal to a vowel I would total ++.

Because my vowels variable looked like this:

const vowels = ['a','e','i','o','u',];

if the user entered a capital letter vowel it would'nt register. I just added the capital letters to the variable like so:

const vowels = ['a','A', 'e','E', 'i','I', 'o','O', 'u','U'];

Although I probably could have used .toLowerCase() somehow.
